# 贡献指南

感谢你愿意贡献！本项目欢迎功能改进、Bug 修复和文档优化。

## 开发环境

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

## 提交规范

- 保持代码风格一致
- 避免引入新依赖（如需引入，请说明原因）
- 不提交个人路径或隐私信息

## 提交流程

1. Fork 本仓库
2. 新建分支：`git checkout -b feat/your-feature`
3. 提交变更：`git commit -m "feat: your change"`
4. 推送：`git push origin feat/your-feature`
5. 提交 PR

## Issue 反馈

请描述：
- 复现步骤
- 期望结果
- 实际结果
- 环境信息（系统版本、Python 版本）

谢谢你的贡献！
